## 🎯 装修效果计算

### 计算逻辑

```typescript
function calculateDecorEffect(decorState: ShopDecorState) {
  let patienceBonus = 0
  let tipBonus = 0
  let scoreBonus = 0

  // 累加所有装修效果
  for (const itemId of [shopFront, table, wall, shelf, cup]) {
    patienceBonus += item.effect.patienceBonus ?? 0
    tipBonus += item.effect.tipBonus ?? 0
    scoreBonus += item.effect.scoreBonus ?? 0
  }

  return { patienceBonus, tipBonus, scoreBonus }
}
```
